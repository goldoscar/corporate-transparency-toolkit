# Methodology

## Objective

This project uses OpenCorporates data to support transparent, reproducible and public-interest company research workflows.

## Planned Workflow

1. Accept a company name, identifier, or jurisdiction as input.
2. Query the OpenCorporates API for matching company records.
3. Normalize and review returned data for consistent entity identification.
4. Link results back to original OpenCorporates records where possible.
5. Produce structured outputs for public-interest research, documentation, or reporting.

## Intended Users

- journalists
- academic researchers
- nonprofit investigators
- students
- civil-society users
- general public users interested in transparency

## Reproducibility

All workflow steps, assumptions, and output formats will be documented publicly in this repository so that others can understand, inspect, and reuse the process.

## Attribution Practice

Any published output that relies on OpenCorporates data will clearly attribute OpenCorporates and, where appropriate, link to the corresponding company page or source reference.
