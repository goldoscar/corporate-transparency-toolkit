# Data Sources

## Primary Data Source

The primary external data source for this project is OpenCorporates.

- Website: https://opencorporates.com
- Intended access method: OpenCorporates API
- Intended use: company lookup, entity verification, structured public-interest research workflows

## Additional Data Sources

At the time of application, this project is not intended to combine OpenCorporates data with another dataset.

If additional datasets are incorporated in the future, their origin, licence terms, and compatibility considerations will be documented here before publication.

## Licensing Note

Any derived database or data output incorporating OpenCorporates data will be handled in a manner consistent with the applicable OpenCorporates open data licence and attribution requirements.
