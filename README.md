# OpenCorporates Public Benefit Tool

A public-benefit open-source project designed to support transparent, reproducible company research workflows using OpenCorporates data.

## Public Benefit Purpose

This project is intended to help researchers, journalists, students, civil-society organisations, and other public-interest users investigate legal entities more easily and consistently.

The goal is to improve transparency by providing a free and openly documented workflow for:

- company lookup
- entity verification
- structured research support
- reproducible public-interest investigation workflows

This project is not intended to be a closed commercial product. The planned outputs will be made publicly available.

## How OpenCorporates Data Will Be Used

The project will use the OpenCorporates API for:

- company search and lookup
- entity verification and consistency checks
- linking findings back to source company records
- supporting transparent and reproducible research workflows

OpenCorporates data will be used as part of a public-interest workflow that will be openly documented in this repository.

## Planned Public Outputs

The following materials will be published openly in this repository:

- source code
- setup instructions
- methodology documentation
- attribution details
- public releases of the tool or research workflow

If appropriate, a public project page or short report may also be published.

## Publication Timeline

Initial public repository materials are available now. The first public release is planned within 6 weeks of API access approval.

## Free Public Access

The service and content produced by this project are intended to be free to use for all.

## Licensing and Attribution

Source code in this repository may be released under a separate open-source software licence.

Any derived database or data output incorporating OpenCorporates data will be published under ODbL / share-alike terms where applicable, with clear attribution to OpenCorporates.

Where company-level information is displayed, the project will link back to the relevant OpenCorporates company page where possible.
